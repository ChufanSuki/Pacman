Usage¶

Installation¶

To use Pacman, first install it using pip:

(.venv) $ pip install pacman

Creating recipes¶

To retrieve a list of random ingredients, you can use the pacman.get_random_ingredients() function:

The kind parameter should be either "meat", "fish", or "veggies". Otherwise, pacman.get_random_ingredients() will raise an exception.

For example:

>>> import pacman
>>> pacman.get_random_ingredients()
['shells', 'gorgonzola', 'parsley']
